## üöÄ How to Play

### For Players (Standalone Executable)
1.  Download the `Pixel Adventurer Game.exe` file.
2.  Double-click the executable to launch the game.
3.  **Controls:**
    * **Arrows / WASD:** Move and Jump.
    * **Space:** Action/Jump.
    * **Esc:** Quit.

### For Developers (Source Code)
1.  Install **Godot 4**.
2.  Clone this repository.
3.  Open Godot and click **Import**.
4.  Navigate to the project folder and select the `project.godot` file.
